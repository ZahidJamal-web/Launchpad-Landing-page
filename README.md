# 🚀 LaunchPad - Modern SaaS Landing Page

A modern, responsive, and elegant SaaS landing page built using **HTML, CSS, and JavaScript**. LaunchPad features a premium user interface with dark/light mode, glassmorphism effects, smooth animations, and a responsive design optimized for all devices.

## 🚀 Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/your-username/launchpad.git
```

### 2. Open Project Folder

```bash
cd launchpad
```

### 3. Run Locally

Simply open:

```text
index.html
```

or

```text
login.html
```

in your browser.

No installation required.
